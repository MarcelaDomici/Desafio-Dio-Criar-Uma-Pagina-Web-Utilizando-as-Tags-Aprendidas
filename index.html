Introdução

O HTML (HyperText Markup Language) é a linguagem de marcação que dá estrutura a praticamente todas as páginas da internet. Cada elemento é definido por tags, que dizem ao navegador como o conteúdo deve ser exibido. A seguir, você vai conhecer as principais tags para começar a construir sites do zero.

Estrutura da página: head, body e title

O esqueleto de qualquer página HTML se divide em duas partes:

  • O head guarda informações que não aparecem diretamente para o usuário, como o título, metadados e links para arquivos externos (CSS ou JavaScript).
  • O title fica dentro do head e define o título mostrado na aba do navegador.
  • O body contém todo o conteúdo visível da página: textos, imagens, links, listas e muito mais.

Sem essas tags, o navegador não conseguiria organizar corretamente as informações.

Títulos e parágrafos: h1, h2 e p

Para organizar o texto, usamos títulos e parágrafos:

  • h1 é o título principal da página (geralmente usado apenas uma vez).
  • h2 indica subtítulos que organizam o conteúdo em seções.
  • p representa os parágrafos, que deixam o texto mais legível e estruturado.

Essa hierarquia é importante também para SEO (otimização para mecanismos de busca).

Ênfase e destaque no texto: strong, i, u e mark

O HTML oferece várias formas de destacar informações:

  • strong deixa o texto em negrito e indica importância.
  • i coloca o texto em itálico, usado para ênfase leve ou termos estrangeiros.
  • u sublinha o conteúdo, chamando a atenção do leitor.
  • mark marca o texto como se fosse um marca-texto, ideal para realçar trechos importantes.

Essas tags são muito úteis em artigos, blogs e páginas de estudo.

Listas: ol, ul e li

As listas organizam informações em sequência:

  • ol cria uma lista ordenada (numerada).
  • ul gera uma lista não ordenada (com marcadores).
  • li representa cada item dentro de uma lista.

Exemplo prático:

  • Lista de compras com ul: pão, leite, café.
  • Passo a passo com ol: ligar o forno, preparar a massa, assar o bolo.

Links e formulários: a e input

A navegação na web só é possível graças a:

  • a, que cria links clicáveis, podendo levar a outra página, parte do mesmo site ou até abrir um e-mail.
  • input, que é usado em formulários, permitindo inserir dados como textos, senhas, botões de envio e seleções.

Essas duas tags são a base da interatividade de um site.

Citações e referências: blockquote

Quando precisamos destacar uma citação, usamos a tag blockquote. Ela cria uma área específica no texto, geralmente com recuo, para mostrar falas de autores, trechos de livros ou referências externas.

Exemplo:

“O HTML é a espinha dorsal da web moderna.”

Isso deixa o conteúdo mais claro e visualmente organizado.

Conclusão

Viu só? Apenas com esse conjunto de tags já é possível montar uma página básica e funcional. É como montar um sanduíche com pão, queijo e presunto: simples, mas delicioso. Com head, body, title e companhia, você já pode dizer: parabéns, você criou sua primeira página na web!
